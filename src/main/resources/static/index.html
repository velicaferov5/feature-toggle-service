Welcome to Feature Toggle API

API allows to insert, get, update, archive and invert feature toggles.

Feature Toggles have: ID DisplayName TechnicalName ExpiresOn Description Inverted CustomerIds

Feature Toggles can be inserted, get, update, archive, inverted with REST services.

Requirement to run (Java version): 11


REST-ful services in API:

Check API
URL: /api/
Input: none
Output: none
Type: Any
Use: To print API is working

FeatureToggles
URL: /api/featuretoggle/{id}
Input: id (int)
Output: FeatureToggle
Type: GET
Use: Get Feature Toggle by ID

URL: /api/featuretoggle/
Input: FeatureToggleDTO
Output: FeatureToggleDTO
Type: POST
Use: To insert new feature toggle

URL: /api/featuretoggle/{id}
Input: header: id (int), body: FeatureToggleDTO
Output: FeatureToggleDTO
Type: PUT
Use: To update a feature toggle

URL: /api/featuretoggle/archive/{id}
Input: id (int)
Output: FeatureToggleDTO
Type: PUT
Use: To archive a feature toggle
